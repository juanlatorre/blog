Bye Pelican, Hola WordPress
1 febrero, 2018 , en Blog

Ha pasado un buen tiempo desde que instalé este blog en Pelican utilizando Python y Github, sin embargo el proceso para publicar nuevas entradas no era lo suficientemente rápido como para quedar satisfecho, así fue como investigando y haciendo zapping digital me encontré con un plugin llamado Simply Static que aclamaba generar una copia exacta del sitio en cuestión como una copia estática, sonaba perfecto para subir a Github Pages, así que me animé a probarlo.

Después de un par de pruebas noté que funcionaba perfecto, basicamente, al apretar el botón de generar (y modificar un poco los ajustes a gusto), se genera una copia estática en la carpeta a elección, así que decidido a utilizar wordpress en un servidor local para generar el blog, me puse manos a la obra “portando” el tema que había construído en Pelican para pasarlo a WordPress, después de un par de ajustes y unas horas de color testing el resultado que más me gustó vio la luz.

Sin embargo tenía ciertos problemas, como era de esperar el funcionamiento de algunos plugins se vio alterado, por ejemplo Contact Form 7 dejó de funcionar, así que comenzó la búsqueda de lograr una forma estática de enviar correos a través de un formulario de contacto web, pero eso queda para otro post.
